Connecting to Web Services

Web services you can connect to include:

  • WFS: https://maps.gcc.tas.gov.au/geoserver/GCC_cc/ows?service=wfs&version=1.1.0&request=GetCapabilities
  • WMS: https://maps.gcc.tas.gov.au/geoserver/GCC_cc/gwc/service/wms?version=1.1.1&request=GetCapabilities
Some documentation for connecting to these can be found here:
  • Leaflet: for leaflet, use a TileLayer (WMS). We have examples in our maps section and you can see all our code on GitHub.
  • MapInfo: see this site: MapInfo WMS Setup for WMS. WFS is similar.
  • QGIS: The above URLs work in QGIS. Try this Spatial Thoughts tutorial.
  • ArcGIS: See the documentation for WMS or WFS.
